This project is totally based on fundamental concepts of programming in C++. It allows user to view the flights schedule , book a ticket ,cancel a ticket. Also allows to login as admin and view the booked tickets , alter the flights schedule
Online Election Portal
GUI project in JAVA. It allows user to register or login into their account and cast a vote to their favorite party and also view the election results. This has huge checks like one voter can't vote twice, one CNIC can be registered only once and much more.
Maze Generation and Path Planning
In this project, I use processing language to generate Maze and then find the shortest path of the Maze. Using DFS method of Graphs , maze is generated. And A* search algorithm finds the shortest path of the Maze.
